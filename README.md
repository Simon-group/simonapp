# README

Moviecollection
====
Overview
パソコンにダウンロードした動画を投稿出来る、動画投稿アプリ
ユーザーの新規登録、ログイン、投稿した動画のマイページ画面の確認が出来る。
詳細画面にて、動画に対していいねやコメントを記述する事も出来る。
